# Instalación
Se recomienta utilizar Virtual Environment

``` Shell
python3.6 -m venv ve_py36_pycon # Puede ser Python 3.8 también
source ve_py36_pycon/bin/activate
pip install pip --upgrade # No olvidar

pip install -r requirements.txt 

```
